## Overview
XeroJS provides a structured API for creating extensions, UI modules, hooks, and runtime logic inside XeroCord.  
It is sandboxed, modular, and fastâ€”built for safe client modding.

---

## Features
- Executes `.xo` scripts  
- Secure sandboxed runtime  
- UI framework for panels, buttons, toggles  
- Message & client event hooks  
- Plugin isolation and error protection  
- Lightweight parser and AST engine
